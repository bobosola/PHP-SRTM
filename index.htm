SRTM Elevation Data in PHP

This is a demo of my PHP code to read v4 SRTM GeoTIFF data files.

You will first need to unzip the SRTM data file in ./GeoData. otherwise the demo will not work. The data covers the UK from 50°N to 55°N and 0° to 5°W (-5°).

Locations within the tile area but over the sea (e.g. 51.3, -4.5) will show as -32768 (the SRTM 'no data' value). Locations out of the tile range will throw an error and die this but can be supressed.

The default values below are for Mount Snowdon, which at 1085m at the peak is the highest mountain inWales (53.0685, -4.0763).
